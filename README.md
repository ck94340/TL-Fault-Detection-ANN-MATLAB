# 🔌 AI-Based Transmission Line Fault Detection using ANN (MATLAB)

## 📌 Overview
This project presents an AI-based approach for detecting and classifying faults in transmission lines using Artificial Neural Networks (ANN) implemented in MATLAB/Simulink.

---

## ❗ Problem Statement
Transmission line faults can lead to power loss and system instability. Traditional methods are slow and less accurate. This project aims to develop a fast and reliable fault detection system using AI.

---

## 🧠 Methodology
- Data collected from simulation of transmission line faults
- Preprocessing of voltage and current signals
- ANN model trained to classify different types of faults
- Simulation performed in MATLAB Simulink

---

## ✨ Features
- Automatic fault detection  
- Fault classification (LG, LL, LLG, LLLG)  
- High accuracy prediction  
- Real-time simulation capability  

---

## 🛠 Tools & Technologies
- MATLAB  
- Simulink  
- Artificial Neural Network (ANN)  

---

## 📊 Results
- Achieved high accuracy in fault classification  
- Fast response time  
- Graphical outputs for voltage and current signals  

---

## ⚡ Applications
- Power system monitoring  
- Smart grid systems  
- Fault diagnosis in electrical networks  

---

## ▶️ How to Run
1. Open MATLAB  
2. Load the Simulink model (.slx file)  
3. Run the simulation  
4. View results in scope/output graphs
